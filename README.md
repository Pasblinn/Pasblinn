Tenho experiência com desenvolvimento de **APIs**, **aplicações web**, **sistemas cliente-servidor**, **automação de processos** e **integração com Inteligência Artificial**. Atuo do backend à interface, com atenção à arquitetura, organização de código, desempenho e experiência de uso.
